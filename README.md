# TodoApp Gemini

Este é um projeto de aplicação de lista de tarefas (TodoApp) completo, construído para demonstrar as capacidades de desenvolvimento de software de um agente de IA (Gemini).

## Tecnologias Utilizadas

- **Frontend:** Vue.js 3, Vite, TypeScript
- **Backend:** Node.js, Express.js
- **Banco de Dados:** SQLite 3

## Funcionalidades

- **Adicionar Tarefas:** Um usuário pode adicionar novas tarefas à lista.
- **Editar Tarefas:** Um usuário pode editar o título de uma tarefa existente com um duplo clique.
- **Marcar como Concluída:** Um usuário pode marcar ou desmarcar uma tarefa como concluída.
- **Remover Tarefas:** Um usuário pode remover tarefas da lista.

## Como Executar o Projeto

Para rodar a aplicação, você precisará de dois terminais abertos.

### 1. Terminal do Backend

Navegue até a pasta `backend` e execute o servidor da API:

```bash
cd backend
npm run dev
```
O servidor estará disponível em `http://localhost:3000`.

### 2. Terminal do Frontend

Navegue até a pasta `frontend` e execute o servidor de desenvolvimento:

```bash
cd frontend
npm run dev
```
O Vite iniciará o servidor e fornecerá uma URL local para acessar a aplicação (geralmente `http://localhost:5173`).
